# Im Nafis — Personal Portfolio

Repository ini berisi source code untuk website personal portfolio Nafis yang berjalan di Next.js, TypeScript, Tailwind CSS, Supabase, dan Vercel.

Website utama: https://im-nafis.vercel.app

## Isi Website

- Profil utama dan CTA kontak
- Track Record: riwayat pendidikan, pencapaian utama, kepemimpinan & organisasi, serta keahlian inti
- Karya pilihan dengan modal detail
- Produk dan layanan digital
- Admin dashboard untuk mengelola konten utama

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase Database & Auth
- Vercel Deployment

## Development

```bash
npm install
npm run dev
```

Validasi sebelum push:

```bash
npm run lint
npm run build
```

## Admin Content

Konten dinamis dikelola dari halaman admin dan tersimpan di Supabase, termasuk profil, social links, portfolio, services, dan track record.
